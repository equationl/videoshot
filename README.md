经常在网上看到别人制作的电影台词的长截图？想要自己制作一张？一张一张截图后剪切再拼接很麻烦？ 

有了视频截取，你就可以边看视频边将想要截图的位置标记下来，当视频播放完后忙你的截图也已经处理完毕。 

可用于制作电影图解 or 电影台词截图 or 精彩画面合集。 

还支持自定义裁剪范围，甚至是保留整张截图。
